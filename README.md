#  SmartHome IoT Simulator

Симулятор системи «Розумний будинок» (IoT Simulator), розроблений на мові C# з використанням платформи .NET 9 та Windows Forms у рамках навчальної практики з об’єктно-орієнтованого програмування.

Проєкт демонструє застосування принципів ООП, SOLID, патернів проєктування, технологій LINQ, механізмів серіалізації даних та юніт-тестування.

---

##  Основні можливості

-  Автоматичне керування пристроями на основі даних сенсорів:
  - температура
  - рух
  - освітлення

-  Ручне керування пристроями:
  - лампа
  - обігрівач
  - кондиціонер

-  Інтерактивні перемикачі ON/OFF (зелений / червоний стан)

-  Розклад роботи лампи (реалізовано через Decorator)

-  Екорежим обігрівача

-  LINQ-звіти:
  - фільтрація
  - групування
  - середнє значення

-  Збереження стану системи:
  - JSON
  - XML

-  Юніт-тестування:
  - xUnit
  - Moq

-  Графічний інтерфейс (Windows Forms)

##  Використані патерни проєктування

У проєкті реалізовано такі патерни:

###  Observer
Сенсори повідомляють контролер про зміну свого стану.

###  Facade
Клас `SmartHomeFacade` надає спрощений інтерфейс для взаємодії з системою.

###  Decorator
Додає додаткову функціональність до пристроїв:
- розклад лампи
- екорежим обігрівача

###  Strategy
Реалізує алгоритми автоматичного керування:
- керування температурою
- реакція на рух
- реакція на освітлення

###  Factory
Відповідає за створення сенсорів та пристроїв.

##  SOLID-принципи

У проєкті дотримано принципів SOLID:

- **SRP** — кожен клас має одну відповідальність  
- **OCP** — систему можна розширювати без зміни існуючого коду  
- **LSP** — похідні класи замінюють базові без помилок  
- **ISP** — інтерфейси розділені за відповідальністю  
- **DIP** — залежності побудовані через інтерфейси
